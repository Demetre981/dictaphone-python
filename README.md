
# Dictaphone Tutorial

## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install PyAudio
pip install tkinter
```

## Running The App

```bash
python main.py 
```

## Using The App

Click on the microphone image and start recording. Intermediary WAV files are saved in the project folder.
